  • There are IRE endpoints that front-end code interfaces with via API/AJAX requests, and those requests are constructed with URI origins, one of which is "https://maint-dev-www.sherwin-williams.com". But naturally, any origin hosting the API can serve as an IRE endpoint context. Scene7 also hosts an API, but it's on a dedicated origin, e.g. "//sherwin.scene7.com"

    You have to go through F5 to iPlanet. iPlanet has to route the request to weblogic. Weblogic has to send the request to URLRewrite.xml. URLRewrite has to evaluate every rule that doesn't match, until it finds the one you put in, then send the redirect.
